## Estrutura da Planilha
A planilha Importação de Usuário - Moodle.xlsx possui 3 abas:
Etapa 01. Preencher informações --- Dados dos funcionários a importar
Etapa 02. Imprimir PDF -- Layout para entrega ao funcionário
Etapa 03. Salvar como CSV -- Visualização do formato de exportação

## Como Funciona
Ao clicar no botão "Armazenar Dados" na planilha, a macro JavaScript executa automaticamente:
1. CSV - utilizado para exportação do sistema.
2. PDF — Entrega ao Funcionário
3.  XLSX — Arquivo de Backup

## Requisitos
WPS Office (versão gratuita com suporte a macros JS)
Acesso ao servidor de rede 
Planilha salva no formato .et (WPS habilitado para macro)
